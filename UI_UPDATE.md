## 🎯 Новые функции

### 1. Copy Config
```javascript
function copyProxyConfig() {
    const config = 'SOCKS5 127.0.0.1 1080';
    navigator.clipboard.writeText(config);
    showToast('✅ Конфигурация скопирована!');
}
```

**Использование:**
- Нажать кнопку "📋 Копировать конфиг"
- Вставить в настройках Telegram

### 2. QR Code Modal
```javascript
function showQRCode() {
    const qrUrl = `https://api.qrserver.com/v1/create-qr-code/?size=300x300&data=${url}`;
    // Показывает модальное окно с QR
}
```

**Использование:**
- Нажать "📱 QR-код"
- Отсканировать телефоном

### 3. Toast Notifications
```javascript
function showToast(message) {
    // Анимированное уведомление сверху
}
```

**Примеры:**
- ✅ "Конфигурация скопирована!"
- ❌ "Не удалось скопировать"
- 🔄 "Обновление..."

### 4. Smart Status
```javascript
function updateHealthStatus(stats) {
    if (wsErrors < 5) → 🟢 "Работает нормально"
    else if (wsErrors < 15) → 🟡 "Работает с проблемами"
    else → 🔴 "Проблемы с подключением"
}
```

## 🎯 Как использовать

### 1. Откройте панель
```
http://192.168.31.227:8080
```

### 2. Отсканируйте QR-код
Наведите камеру телефона на QR-код в приложении.

### 3. Установите PWA
- **Android:** Меню (⋮) → "Установить приложение"
- **iOS:** Поделиться (📤) → "На экран «Домой»"

### 4. Настройте тему
Нажмите 🌙/☀️ в правом верхнем углу.

### 5. Используйте
- 🔄 Обновляйте статистику
- 📋 Копируйте конфиг
- 📱 Показывайте QR-код
- 🔓 Открывайте в Telegram
