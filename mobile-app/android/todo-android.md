## Новый план (Advanced Optimization):
1.  [ ] **Battery Watchdog**: Авто-выключение прокси при низком заряде (<15%) для спасения устройства.
2.  [ ] **SOCKS5 Auth**: Поддержка установки логина/пароля для прокси-сервера через UI.
3.  [ ] **Traffic Quota**: Возможность установить лимит трафика (на день/месяц).
4.  [ ] **Per-App Proxy (VpnService)**: Исследование и прототип запуска прокси как локального VPN для работы без настроек в приложениях.
5.  [ ] **Network-Aware Pooling**: Умное управление пулом WebSocket соединений в зависимости от типа сети (Wi-Fi/Mobile).
6.  [ ] **Daily Statistics**: Сохранение истории использования трафика за последние 30 дней в локальную БД.
7.  [ ] **JSON Export/Import**: Возможность делиться всеми настройками прокси через один файл.
8.  [ ] **Heartbeat Mechanism**: Двусторонняя проверка связи между JS и Python для обнаружения зависаний.
9.  [ ] **Custom MTProto Secret**: Возможность смены секретного ключа MTProto через настройки.
10. [ ] **UI Upgrade**: Добавление нативных Material 3 анимаций при переключении состояний.
