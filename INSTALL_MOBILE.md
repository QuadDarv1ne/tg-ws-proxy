## Способ 1: PWA (Быстрая установка за 2 минуты) ⚡

**PWA (Progressive Web App)** — это современная технология, позволяющая устанавливать веб-приложения как обычные приложения.

### ✅ Преимущества PWA

- ⚡ **Мгновенная установка** — не нужно ждать сборку
- 🔄 **Автообновление** — всегда последняя версия
- 💾 **Экономия места** — ~1 МБ вместо ~50 МБ
- 🌐 **Кроссплатформенность** — работает на Android и iOS
- 🔒 **Безопасность** — не требует разрешений на установку APK

### ⚠️ Недостатки

- Требуется запущенный веб-сервер на компьютере
- Работает только в локальной сети

---

### 📲 Установка на Android (Chrome)

#### Шаг 1: Запустите прокси на компьютере

```bash
cd C:\Users\maksi\OneDrive\Documents\GitHub\tg-ws-proxy
python run_web.py
```

Ожидайте сообщение:
```
Starting web dashboard on http://127.0.0.1:5000
```

#### Шаг 2: Узнайте IP-адрес компьютера

**Windows:**
```bash
ipconfig
```

Найдите строку **IPv4 Address** в разделе вашей сети (Wi-Fi или Ethernet):
```
IPv4-адрес. . . . . . . . . . . . : 192.168.1.100
```

**Запомните этот адрес!**

#### Шаг 3: Откройте на телефоне

1. **Подключите телефон к той же Wi-Fi сети**, что и компьютер
2. **Откройте Chrome на телефоне**
3. **Введите адрес:**
   ```
   http://192.168.1.100:5000
   ```
   (замените на ваш IP из Шага 2)

#### Шаг 4: Установите приложение

**Вариант A — Автоматическая установка:**
1. Внизу экрана появится баннер: **"Установить TG WS Proxy?"**
2. Нажмите **"Установить"**

**Вариант B — Через меню:**
1. Нажмите **⋮** (три точки в правом верхнем углу)
2. Выберите **"Установить приложение"** или **"Добавить на главный экран"**
3. Нажмите **"Установить"** или **"Добавить"**

#### Шаг 5: Запустите

1. Найдите иконку **TG WS Proxy** на главном экране или в меню приложений
2. Нажмите для запуска
3. Готово! 🎉

## Способ 2: APK (Полноценное приложение) 📦

**APK** — это установочный файл Android приложений. Требует сборки.

### ✅ Преимущества APK

- 📱 **Работает без компьютера** — приложение автономно
- 🚀 **Быстрый запуск** — не нужно ждать загрузки веб-страницы
- 🔔 **Уведомления** — полноценная поддержка push-уведомлений
- 📶 **Офлайн-режим** — работает без подключения к компьютеру

### ⚠️ Недостатки

- Требуется **Android Studio** (~3 ГБ)
- Сборка занимает **10-20 минут**
- Нужно **обновлять вручную** при выходе новых версий

---

### 🛠️ Требования для сборки APK

| Компонент | Версия | Где взять |
|-----------|--------|-----------|
| **Android Studio** | 2024.1+ | https://developer.android.com/studio |
| **Java JDK** | 17+ | Встроено в Android Studio |
| **Android SDK** | 34+ | Устанавливается со Studio |
| **Node.js** | 18+ | https://nodejs.org (уже установлен) |
| **Свободное место** | ~5 ГБ | На диске C: |

---

### 📥 Шаг 1: Установка Android Studio

#### 1.1 Скачайте

1. Перейдите на https://developer.android.com/studio
2. Нажмите **"Download Android Studio"**
3. Примите условия лицензии
4. Скачается файл ~1.2 ГБ (например, `android-studio-2024.2.1.12-windows.exe`)

#### 1.2 Установите

1. **Запустите установщик**
2. **Нажмите "Next"**
3. **Выберите компоненты:**
   - ✅ Android Studio
   - ✅ Android Virtual Device (можно не ставить для экономии места)
4. **Выберите папку установки** (по умолчанию `C:\Program Files\Android\Android Studio`)
5. **Нажмите "Install"**
6. **Дождитесь установки** (5-10 минут)

#### 1.3 Первый запуск

1. **Запустите Android Studio**
2. **Нажмите "Do not import settings"** → **OK**
3. **Выберите тип установки:** **Standard**
4. **Выберите тему:** **Dark** или **Light**
5. **Нажмите "Finish"**
6. **Дождитесь загрузки компонентов** (10-15 минут)

### ⚙️ Шаг 2: Настройка переменных среды

#### 2.1 Узнайте путь к SDK

1. В Android Studio откройте **Tools** → **SDK Manager**
2. Вверху увидите путь, например:
   ```
   C:\Users\maksi\AppData\Local\Android\Sdk
   ```

#### 2.2 Добавьте переменные среды

**Windows 10/11:**

1. Нажмите **Win + R**
2. Введите: `sysdm.cpl`
3. Перейдите на вкладку **"Дополнительно"**
4. Нажмите **"Переменные среды"**
5. В разделе **"Переменные среды пользователя"** нажмите **"Создать"**

**Создайте переменную ANDROID_HOME:**
- **Имя:** `ANDROID_HOME`
- **Значение:** `C:\Users\maksi\AppData\Local\Android\Sdk`

**Добавьте в PATH:**

1. Найдите переменную **Path** в списке
2. Выделите её и нажмите **"Изменить"**
3. Нажмите **"Создать"** и добавьте:
   ```
   %ANDROID_HOME%\platform-tools
   %ANDROID_HOME%\cmdline-tools\latest\bin
   %ANDROID_HOME%\tools\bin
   ```
4. Нажмите **OK** во всех окнах

#### 2.3 Проверка

Откройте **новую** командную строку и проверьте:

```bash
adb --version
```

Должно вывести версию ADB.

#### 4.1 Подготовьте проект




#### Вариант C: Через Wi-Fi

### 🔐 Шаг 6: Подписание APK (для релиза)

Для установки на другие устройства нужно подписать APK:

#### 6.1 Создайте ключ

```bash
keytool -genkey -v -keystore tg-ws-proxy.keystore -alias tg-ws-proxy -keyalg RSA -keysize 2048 -validity 10000
```

Введите:
- Пароль ключа
- Имя: **Dupley Maxim Igorevich**
- Организация: (по желанию)
- Код страны: **RU**

#### 6.2 Настройте сборку

Откройте `mobile-app/android/app/build.gradle`:

```gradle
android {
    ...
    signingConfigs {
        release {
            storeFile file('tg-ws-proxy.keystore')
            storePassword 'ваш_пароль'
            keyAlias 'tg-ws-proxy'
            keyPassword 'ваш_пароль'
        }
    }
    buildTypes {
        release {
            signingConfig signingConfigs.release
        }
    }
}
```

#### 6.3 Соберите релизный APK

```bash
gradlew assembleRelease
```

APK будет в: `android/app/build/outputs/apk/release/app-release.apk`

## Устранение проблем

### ❌ PWA не устанавливается

**Проблема:** В Chrome нет опции "Установить приложение"

**Решение:**
1. Убедитесь, что используете **HTTPS** или **localhost**
2. Для локальной сети добавьте исключение:
   - Chrome: `chrome://flags/#unsafely-treat-insecure-origin-as-secure`
   - Включите и добавьте ваш IP: `http://192.168.x.x:5000`
   - Перезапустите Chrome

---

### ❌ Не могу подключиться к компьютеру

**Проблема:** Телефон не видит компьютер в сети

**Решение:**

1. **Проверьте брандмауэр:**
   ```bash
   # Разрешите порт 5000
   netsh advfirewall firewall add rule name="TG WS Proxy" dir=in action=allow protocol=TCP localport=5000
   ```

2. **Проверьте, что компьютер и телефон в одной сети**

3. **Попробуйте отключить VPN** на компьютере

4. **Проверьте IP:**
   ```bash
   ipconfig
   ```
   Убедитесь, что используете правильный IP

---

### ❌ APK не устанавливается

**Проблема:** "Установка заблокирована" или "Неизвестный источник"

**Решение:**

1. **Разрешите установку из неизвестных источников:**
   - **Настройки** → **Безопасность** → **Неизвестные источники**
   - Или при установке нажмите **"Разрешить"**

2. **Проверьте версию Android:**
   - Требуется Android 8.0+

3. **Очистите кэш Google Play:**
   - **Настройки** → **Приложения** → **Google Play** → **Хранилище** → **Очистить кэш**

---

### ❌ Сборка APK не удаётся

**Ошибка:** `SDK location not found`

**Решение:**
1. Откройте `mobile-app/android/local.properties`
2. Добавьте строку:
   ```
   sdk.dir=C\:\\Users\\maksi\\AppData\\Local\\Android\\Sdk
   ```

**Ошибка:** `Java not found`

**Решение:**
1. Убедитесь, что JDK установлен
2. Добавьте переменную `JAVA_HOME`:
   - **Значение:** `C:\Program Files\Android\Android Studio\jbr`

---

### ❌ Прокси не работает

**Проблема:** Telegram не подключается

**Решение:**

1. **Проверьте, что прокси запущен:**
   ```bash
   python run_web.py
   ```

2. **Проверьте порт:**
   ```bash
   netstat -an | findstr 1080
   ```

3. **Перезапустите прокси:**
   - Закройте приложение
   - Запустите снова

4. **Проверьте логи:**
   - Откройте `proxy/proxy.log`
   - Ищите ошибки
