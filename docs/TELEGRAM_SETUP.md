# 📱 Настройка Telegram Desktop

## 🖥️ Windows

### Автоматическая настройка

1. Запустите `tray.py` или `TgWsProxy.exe`
2. ПКМ по иконке в трее
3. **Открыть в Telegram**
4. Telegram автоматически применит настройки прокси

### Ручная настройка

1. Откройте **Telegram Desktop**
2. Нажмите **☰** (меню)
3. **Настройки** → **Продвинутые**
4. **Тип подключения** → **Использовать прокси**
5. Выберите **SOCKS5**
6. Заполните:
   ```
   Хост: 127.0.0.1
   Порт: 1080
   Логин: (оставьте пустым)
   Пароль: (оставьте пустым)
   ```
7. ✅ **Сохранить**

## 🐧 Linux

### Автоматическая настройка

```bash
python3 linux.py
```

В меню выберите **Open in Telegram**

### Ручная настройка

1. Telegram Desktop
2. **Settings** → **Advanced**
3. **Connection Type** → **Use proxy**
4. **SOCKS5**
5. Введите:
   - **Host:** `127.0.0.1`
   - **Port:** `1080`
6. **Save**

## 📱 Мобильные устройства

### Android

1. Установите приложение **Proxy Droid** или **Postern**
2. Добавьте новый прокси:
   - **Type:** SOCKS5
   - **Host:** IP вашего компьютера
   - **Port:** 1080
3. Включите прокси
4. Откройте Telegram
5. Telegram автоматически использует системный прокси

### iOS

1. Установите **Potatso** или **Shadowrocket**
2. Добавьте конфигурацию:
   - **Type:** SOCKS5
   - **Server:** IP компьютера
   - **Port:** 1080
3. Включите в системные настройки
4. Откройте Telegram

## 🔧 Расширенные настройки

### Использование с авторизацией

Если прокси настроен с авторизацией:

```json
{
  "security": {
    "auth_required": true,
    "auth_username": "user",
    "auth_password": "pass"
  }
}
```

В Telegram укажите:
- **Логин:** `user`
- **Пароль:** `pass`

### Несколько прокси

Telegram поддерживает несколько прокси:

1. Настройте первый прокси (127.0.0.1:1080)
2. Добавьте второй через настройки
3. Telegram будет переключаться автоматически

## 🐛 Решение проблем

### Telegram не видит прокси

1. Убедитесь что прокси запущен
2. Проверьте порт (по умолчанию 1080)
3. Проверьте брандмауэр

### Медленная загрузка

1. Проверьте скорость интернета
2. Попробуйте другой DC в настройках прокси
3. Включите compression в конфиге

### Прокси отключается

1. Проверьте логи прокси
2. Увеличьте timeout в конфиге
3. Проверьте стабильность соединения
