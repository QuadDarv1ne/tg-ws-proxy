## Способ 1: PWA (2 минуты) ⚡

**Преимущества:**
- Мгновенная установка
- Автообновление
- ~1 МБ места
- Работает на Android и iOS

**Недостатки:**
- Требуется запущенный сервер на компьютере
- Работает в локальной сети

### Android (Chrome)

1. **Запустите прокси на компьютере:**
   ```bash
   python run_web.py
   ```

2. **Узнайте IP компьютера:**
   ```bash
   ipconfig
   # Найдите IPv4-адрес, например: 192.168.1.100
   ```

3. **Откройте на телефоне:**
   - Подключите телефон к той же Wi-Fi сети
   - Откройте Chrome: `http://192.168.1.100:5000`

4. **Установите:**
   - Нажмите ⋮ (три точки) → **"Установить приложение"** или **"Добавить на главный экран"**
   - Нажмите **"Установить"**

### iOS (Safari)

1. **Запустите прокси:** `python run_web.py`

2. **Узнайте IP:** `ipconfig`

3. **Откройте на iPhone/iPad:**
   - Safari: `http://ВАШ_IP:5000`

4. **Добавьте на главный экран:**
   - Поделиться 📤 → **"На экран «Домой»"** → **"Добавить"**

## Способ 2: APK (Android) 📦

**Преимущества:**
- Работает без компьютера
- Полноценные уведомления
- Офлайн-режим

**Недостатки:**
- Требуется Android Studio (~3 ГБ)
- Сборка 10-20 минут

### Требования
- Android Studio 2024.1+
- Node.js 18+
- JDK 17+ (встроен в Android Studio)
- ~5 ГБ места

### Установка Android Studio

1. Скачайте: https://developer.android.com/studio
2. Установите (Standard установка)
3. Запустите, дождитесь загрузки компонентов

### Настройка переменных среды

1. **Узнайте путь к SDK:**
   - Android Studio → Tools → SDK Manager
   - Путь: `C:\Users\...\AppData\Local\Android\Sdk`

2. **Добавьте переменные среды:**
   - Win+R → `sysdm.cpl` → Дополнительно → Переменные среды
   - Создайте `ANDROID_HOME` = `C:\Users\...\AppData\Local\Android\Sdk`
   - Добавьте в Path:
     - `%ANDROID_HOME%\platform-tools`
     - `%ANDROID_HOME%\cmdline-tools\latest\bin`

3. **Проверка:**
   ```bash
   adb --version
   ```

### Сборка APK

```bash
cd mobile-app
npm install
npx cap sync android
cd android
gradlew assembleDebug
```

**APK будет в:** `mobile-app/android/app/build/outputs/apk/debug/app-debug.apk`

### Установка на телефон

**Вариант A — Через USB:**
1. Включите отладку по USB (Настройки → Для разработчиков)
2. Подключите телефон
3. ```bash
   adb install app-debug.apk
   ```

**Вариант B — Через файловый менеджер:**
1. Скопируйте APK на телефон
2. Откройте и установите

## Настройка прокси в Telegram

### Автоматически
1. Откройте TG WS Proxy на телефоне
2. Нажмите **"Открыть в Telegram"**

### Вручную
1. Telegram → Настройки → Данные и память → Использовать прокси
2. **Тип:** SOCKS5
3. **Сервер:** IP компьютера (например, `192.168.1.100`)
4. **Порт:** `1080`

## Устранение проблем

**PWA не устанавливается:**
- Используйте HTTPS или localhost
- Для локальной сети: `chrome://flags/#unsafely-treat-insecure-origin-as-secure`

**Не могу подключиться к компьютеру:**
- Проверьте брандмауэр:
  ```bash
  netsh advfirewall firewall add rule name="TG WS Proxy" dir=in action=allow protocol=TCP localport=5000
  ```
- Убедитесь, что телефон и компьютер в одной сети

**APK не устанавливается:**
- Разрешите установку из неизвестных источников

**Сборка не удаётся:**
- Создайте `mobile-app/android/local.properties`:
  ```
  sdk.dir=C\:\\Users\\...\\AppData\\Local\\Android\\Sdk
  ```
