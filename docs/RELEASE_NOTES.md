## v2.5.5

**Улучшения:**
- 🌐 Веб-панель управления: улучшенный health check с диагностикой по DC
- 📊 Расширенная статистика: эффективность пула WebSocket, процент ошибок по DC
- 🔧 Обработка ошибок: улучшена обработка ошибок в методах автостарта
- 🛡️ Безопасность: проверка прав администратора для автозапуска Windows
- ⚡ Производительность: оптимизирована функция `_human_bytes()`
- 📝 Логирование: детальное логирование в `_pipe_passthrough()`

**Исправления:**
- Улучшена обработка PermissionError в _set_autostart_windows()
- Добавлено логирование успеха/ошибки для всех платформ автостарта
- Улучшен UI health check с отображением статуса по каждому DC
